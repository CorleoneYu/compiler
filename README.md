# 编译原理

## 编译流程

1. 词法分析
2. 语法分析
3. 语义分析
4. 中间代码生成
5. 代码优化
6. 后端：目标代码生成

## 知识点

1. 文法 语言
2. 正规集 -> （正规式 正规文法 DNA） NFA
3. 自顶向下：LL(1) -> 提取左公因子 消除左递归 FIRST FOLLOW SELECT 预测表
4. 自底向上：LR(1)
5. 运行时存储管理：活动记录 DISPLAY表
